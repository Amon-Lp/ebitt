## 🚀 Usage

### Local Development
1. Open `index.html` in a modern web browser
2. Or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve
```

## 🎯 Customization

### Update Content
- Edit `index.html` to modify text content
- Update contact information in the Contact section
- Add more products in the Products section
- Update news items in the News section

### Modify Colors
Edit CSS custom properties in `styles.css`:
```css
:root {
    --color-primary: #00d4ff;  /* Change main accent */
    --color-secondary: #7b2fff; /* Change secondary */
    /* ... */
}
```

### Add Images
Replace the SVG placeholders with actual product images:
```html
<div class="category-header">
    <img src="path/to/product-image.jpg" alt="Product Name" class="product-image">
    <!-- ... -->
</div>
```
